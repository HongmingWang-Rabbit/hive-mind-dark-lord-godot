# 7-Day Jam Scope

## Priority: Core Loop First
Get corruption spreading → essence flowing → minions fighting → win/lose working.

## Day-by-Day Targets

### Days 1-2: Foundation (CURRENT)
- [x] Project setup, autoloads
- [x] TileMap world generation
- [x] Corruption spread mechanic
- [ ] Basic HUD showing essence/corruption
- [ ] Click to manually spread corruption (debug)

### Days 3-4: Entities
- [ ] Minion scene (CharacterBody2D + sprite)
- [ ] Basic movement (click to move or auto-patrol)
- [ ] Human scene (civilians that flee)
- [ ] Combat: minions kill humans on contact
- [ ] Essence gain on kill

### Days 5-6: Buildings & AI
- [ ] Portal building (dark lord base)
- [ ] Spawning pit (click to spawn minions)
- [ ] Corruption node (auto-spreads nearby)
- [ ] Enemy spawning at threat thresholds
- [ ] Police/military AI (attack minions)

### Day 7: Polish & Win/Lose
- [ ] Win screen at 80%
- [ ] Lose screen if essence depleted
- [ ] Sound effects (optional)
- [ ] Title screen (optional)
- [ ] Balance tuning

## Minimum Viable Product
1. Map with corruption overlay
2. Essence that ticks up/down
3. Spawnable minions
4. Killable humans
5. Win at 80% / Lose at 0 essence

## Cut If Running Out of Time
- Multiple minion types (just use Crawler)
- Buildings (just have passive corruption spread)
- Threat escalation (just one enemy type)
- Camera controls
- Sound/music
- Menus

## Stretch Goals (Post-Jam)
- Possession mechanic (convert humans)
- Building upgrades
- Multiple maps/levels
- Save/load
- Pathfinding for units
